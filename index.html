Deployability and scale

Production path uses government systems plus Google Cloud services.

Capture

WhatsApp/IVR/PWA, register photo OCR, e-Aushadhi/DVDMS import, HMIS aggregate reconciliation.

AI layer

Speech-to-text, translation, OCR, LLM extraction, anomaly explanation and confidence scoring.

Forecasting

Gradient boosting for mature data, seasonal baselines for sparse PHCs, Croston/TSB for intermittent medicines.

Optimization

District graph with facility nodes, road-cost edges, FEFO lots, cold-chain constraints and min-cost flow.

Operations

Cloud Run APIs, Cloud SQL/PostGIS, Pub/Sub ingestion, Cloud Scheduler jobs, Looker/BigQuery analytics.

Privacy

Aggregate counts only, no ABHA or patient PII, raw images deleted after extraction, role-based district access.
